Nowe możliwości CSS w porównaniu z HTML

  1. Responsywne projektowanie (media queries).
  2. Zaawansowane efekty wizualne (cienie, gradienty, animacje).
  3. Lepsza kontrola nad układem (Flexbox, Grid).
  4. Możliwość stosowania zmiennych CSS.
  5. Łatwiejsze zarządzanie stylem na dużą skalę.

Wady CSS

  1. Problemy z kompatybilnością w różnych przeglądarkach.
  2. Złożoność zarządzania dużymi arkuszami stylów.
  3. Brak pełnej izolacji styli między komponentami.
  4. Nieintuicyjna kolejność dziedziczenia i priorytetów.
  5. Wymaga dodatkowej nauki w porównaniu do prostego HTML.
