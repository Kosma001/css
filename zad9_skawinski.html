1. Składnia teoretyczna

<link rel="stylesheet" type="text/css" href="style.css">

2. Gdzie wstawiany jest styl zewnętrzny?

Styl zewnętrzny wstawiany jest w sekcji <head> dokumentu HTML za pomocą znacznika <link>. Znacznik ten może występować wielokrotnie w dokumencie, co pozwala na podłączenie wielu arkuszy stylów CSS.

3. Najprostsza forma wstawiania stylu zewnętrznego

<link rel="stylesheet" href="style.css">

4. Ile arkuszy styli można dołączyć?

Można dołączyć dowolną liczbę arkuszy stylów, jednak w przypadku większej liczby należy zachować zasadę kaskadowości i priorytetu styli.

5. Co się dzieje w przypadku konfliktów?

Jeśli te same style są definiowane w różnych arkuszach, przeglądarka stosuje zasadę kaskadowości – pierwszeństwo mają style zdefiniowane później lub o większej specyficzności.

6. Tworzenie i edytowanie plików CSS w edytorze tekstowym

Pliki CSS można tworzyć i edytować w dowolnym edytorze tekstowym, zapisując je z rozszerzeniem .css.

7. Tworzenie i edytowanie plików CSS w specjalnym edytorze

Specjalne edytory CSS, takie jak Adobe Dreamweaver czy Visual Studio Code, umożliwiają podgląd i automatyczne formatowanie kodu.

8. Jakie rozszerzenie posiada plik CSS?

Plik będący zewnętrznym arkuszem styli posiada rozszerzenie .css.

9. Zalety zewnętrznego arkusza styli

  • 1. Ułatwia zarządzanie wyglądem wielu stron naraz.
  • 2. Pozwala na szybsze ładowanie stron poprzez cache przeglądarki.

10. Uwagi na temat pisania w CSS

Podczas pisania CSS należy unikać nadmiernej specyficzności selektorów, stosować jednolitą konwencję nazw oraz organizować kod w logiczne sekcje.
